## What It's about

It's about a development platform that enables you to create web applications with these functionalities:
- Secure login access using email and password;
- Password provided via email invitation;
- 3 user types: administrator, application owner, and invited user;
- Access log;
- Storage of various data types (strings, integers, decimals, dates,
etc.);
- Data display via forms and lists;
- Data access shared between different users;
- Restrict data access and management via permissions and
roles;
- Data export.
