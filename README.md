## What It's about

It's about a Web Application that enables an Application Owner to share data with Authorized Invited Users.

### Design your application

Accessing as Administrator you can define which data the Application Owner can share with Users and which operations the Users can do on the data shared.

You can design the application via a Resources Layer and a Presentation Layer and you can create Roles that define what a Users can do.

### Share Data with Users

Accessing as Owner you can Create, Read, Update and Delete Data to share with User. 

### Invite Users to use the application

Accessing as Owner you should create Users and enable them to use the application in a role you want.

When you create a user an email was sent him with a temporary password to access the application. After that you should enable the user with a role.

### What the User can do

User can Create, Read, Update and Delete on Data shared. 

### An example case 

An application for project managers that wants to monitor activities and sub activities of their projects, Employees can read activities and can update only the work status of sub activities.
